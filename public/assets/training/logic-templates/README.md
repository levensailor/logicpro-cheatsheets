## Screenshot Guidelines

- **Resolution:** 2880x1800 or higher (Retina display quality)
- **Format:** PNG with transparency where appropriate
- **Content:** Actual Logic Pro interface screenshots showing realistic, professional session layouts
- **Annotations:** Use arrows, labels, or highlights to draw attention to key elements
- **Consistency:** Use the same Logic Pro theme/appearance across all screenshots
- **Clarity:** Ensure text is readable and interface elements are clearly visible

## Sourcing Screenshots

1. **Create from scratch:** Build the rock band template in Logic Pro following the lesson steps, take screenshots at each stage
2. **Find existing:** Search for Creative Commons or royalty-free Logic Pro session screenshots online
3. **Commission:** Hire a Logic Pro user to create the specific session layouts and capture screenshots
4. **Use diagrams:** For conceptual steps (like Step 1), create illustrated diagrams showing benefits rather than direct screenshots
