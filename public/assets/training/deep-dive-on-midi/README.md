## Image Guidelines

When creating/capturing screenshots:
- Use Logic Pro's default theme for consistency
- Capture at high resolution (at least 1920px wide for screenshots)
- Annotate important areas with arrows or highlights
- Use consistent color coding across related images
- Include both wide views (full interface) and zoomed details
- Show real-world equipment in well-lit, professional photos
